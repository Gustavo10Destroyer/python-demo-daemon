## Como usar?

### Verificar estado do daemon

```sh
python main.py # Mostra o estado do daemon
```

### Iniciar o daemon

```sh
python main.py start # Inicia o daemon (caso não esteja em execução)
```

### Parar o daemon

```sh
python main.py stop # Para o daemon (caso esteja em execução)
```
